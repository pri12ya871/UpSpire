TypeError: Cannot read property 'filter' of undefined

It looks like you're trying to use the filter method on a variable that is undefined. This usually happens when your data hasn't loaded yet or wasn't initialized properly.

Try adding a check before filtering:

const results = data ? data.filter(item => item.name.includes(searchTerm)) : [];
